1-O QUE SÃO DADOS?

DADOS SÃO UM FATO BRUTO, PORÉM DELE PODEMOS EXTRAIR INFORMAÇÕES CONCRETAS SEM SEREM VAZIAS.

2-O QUE SÃO INFORMAÇÕES?

SÃO DADOS ORGANIZADOS SOBRE ALGO OU ALGUÉM, SENDO ALGO CONCRETO.

3-O QUE É UM BANCO DE DADOS?

É UM CONJUNTO DE INFORMAÇÕES ORGANIZADAS, CONTENDO VARIOS DADOS SOBRE O QUE SE FOI ARMAZENADO.
